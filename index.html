Projekty – wizytówka inżynierska

Każdy projekt demonstruje architekturę, przepływ danych i decyzje techniczne – pokazuję, jak myślę i co potrafię zbudować.

CloudBudget – FinOps

Platforma do agregacji kosztów chmurowych z prognozowaniem trendów. Architektura: FastAPI (gateway), Celery + RabbitMQ (kolejkowanie zadań), DuckDB (analiza OLAP). Dane źródłowe pobierane przez skrypty scraping/csv, wyniki przechowywane w PostgreSQL. Model regresji liniowej do predykcji wydatków.

⚙️ Jak działa?

Użytkownik przesyła plik z danymi kosztów → API weryfikuje i wysyła zadanie do Celery → worker wykonuje agregację w DuckDB → wyniki zapisywane w DB → endpoint zwraca raport JSON/CSV. Skalowalne dzięki asynchroniczności.

InfraFlow – Observability

System monitorowania infrastruktury zbierający metryki CPU, RAM, ruch sieciowy. Architektura: Python agents (push co 30s) → FastAPI ingest → Redis Streams bufor → TimescaleDB (hypertables). Dashboard Grafana z predefiniowanymi panelami alertów. Obsługa > 1000 zdarzeń/s.

📡 Przepływ danych

Agent wysyła POST /metrics → API zapisuje w Redis (kolejka) → konsument zapisuje do TimescaleDB po batch. Wizualizacja w czasie rzeczywistym przez WebSocket.

NetGuardian – SecOps

Platforma do detekcji anomalii w logach bezpieczeństwa. Strumienie zdarzeń przez Apache Kafka → silnik reguł (YARA‑like) + analiza behawioralna. Przechowuje zdarzenia w Elasticsearch (opcjonalnie). Alerty wysyłane przez WebSocket do frontu demo.

🧠 Inteligencja

Reguły konfigurowalne przez API, wykrywanie nieudanych logowań, skanowanie portów, podejrzane procesy. Raporty okresowe w PDF.
KafkaPython asyncioElasticsearchWebSocket
Repozytorium

NetAegis – Orchestration MCP

Wieloagentowy system do automatyzacji konfiguracji sieci (Model‑Controller‑Protocol). Agenci: NetPulse (telemetria), SecLog (audyt), NetConfig (push konfiguracji). Orkiestrator napisany w FastAPI + Redis Queue do zadań asynchronicznych. Idempotentne zmiany i historie wersji.

🔁 Przepływ

Operator wywołuje endpoint /deploy → zadanie w RQ → agent NetConfig łączy się przez SSH z urządzeniem i stosuje konfigurację. Wszystkie zmiany logowane.
