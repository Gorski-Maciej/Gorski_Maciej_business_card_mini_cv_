CloudBudget

FinOps – Analityka kosztów chmury

FastAPI → Celery+RabbitMQ → DuckDB → PostgreSQL

System do agregacji kosztów z wielu źródeł (AWS, Azure, API), prognozowania trendów i automatycznego generowania rekomendacji oszczędności. Wykorzystuje asynchroniczne kolejki zadań oraz kolumnową bazę DuckDB do szybkiej analityki.

  • Silnik analityczny: DuckDB – kolumnowa baza dla dużych zbiorów danych finansowych.
  • Harmonogram zadań: Celery Beat + RabbitMQ – cykliczne pobieranie danych i przeliczanie prognoz.
  • Dane operacyjne: PostgreSQL – przechowywanie użytkowników, konfiguracji i historii.
  • API: FastAPI – RESTowe endpointy z walidacją Pydantic i dokumentacją Swagger.

InfraFlow

Obserwowalność i automatyzacja operacyjna

Agenci → FastAPI → TimescaleDB → Grafana

Platforma do zbierania metryk hostów (CPU, RAM, dysk) i zdarzeń infrastrukturalnych. Wyposażona w dashboard operacyjny oraz integrację z Grafana i Prometheus. Redis pełni rolę brokera zadań, zapewniając niezawodność przy dużym obciążeniu.

  • Metryki szeregów czasowych: TimescaleDB – wydajne zapytania o zakresy czasu.
  • Alerting: Konfigurowalne progi – CPU > 90%, brak odpowiedzi hosta.
  • Dashboard: Wizualizacja stanu hostów w czasie rzeczywistym.
  • Docker Compose: Pełna izolacja: API, worker, Redis, TimescaleDB, Grafana.

NetGuardian

SecOps – Detekcja i reakcja na incydenty sieciowe

Kafka → Silnik reguł → Reakcja

System ingestujący zdarzenia przez Apache Kafka, analizujący je behawioralnie i regułowo, a następnie automatycznie uruchamiający reakcje (blokada IP, alert). Wbudowany dashboard pokazuje metryki operacyjne, a opcjonalna baza GeoIP umożliwia geolokalizację zagrożeń.

  • Ingest zdarzeń: Apache Kafka – wysoka przepustowość i skalowalność.
  • Silnik reguł: Analiza behawioralna + reguły statyczne.
  • Automatyczna reakcja: Blokada IP przez API firewalla lub iptables.
  • Geolokalizacja: Opcjonalna baza MaxMind GeoIP.

NetAegis

Orkiestracja bezpieczeństwa – system MCP

MCP Main → MCP Ops → Agenci

Dwuwarstwowy system MCP (Main Control Panel + Operational Panel) z trzema wyspecjalizowanymi agentami: NetPulse (telemetria urządzeń), SecLog (logi bezpieczeństwa) i NetConfig (automatyka konfiguracji). Całość wspiera operatorów w zarządzaniu incydentami i politykami bezpieczeństwa.

  • Architektura MCP: Warstwa główna (zarządzanie) + warstwa operacyjna (wykonawcza).
  • Agenci: NetPulse (SNMP/ICMP), SecLog (syslog), NetConfig (SSH/API).
  • Baza: SQLite dla stanu agentów, Redis dla komunikacji.
  • Polityki: Reguły reagowania definiowane w panelu głównym.
