Flagowe projekty — moja wizytówka

Każdy projekt demonstruje architekturę klasy produkcyjnej, praktyki DevOps i skomplikowane przepływy danych.

CloudBudget – FinOps Platform

Asynchroniczny agregator kosztów z wieloma źródłami danych. Wykorzystuje FastAPI jako API Gateway, Celery + RabbitMQ do kolejkowania zadań oraz DuckDB do szybkiej analizy OLAP. Prognozowanie trendów za pomocą modeli regresji liniowej. Backend przechowuje metadane w PostgreSQL, a raporty generowane są w formacie CSV/JSON.

PythonFastAPICeleryDuckDBPostgreSQL
Repozytorium

InfraFlow – Observability Stack

System monitorowania infrastruktury zbierający metryki w czasie rzeczywistym. Architektura oparta na TimescaleDB (baza szeregów czasowych) oraz Redis Streams do buforowania zdarzeń. Integracja z Grafana i Prometheus. Dashboard pokazuje użycie CPU, RAM, ruch sieciowy i alerty. Własny agent napisany w Pythonie wysyła dane poprzez REST API.

FastAPITimescaleDBRedisGrafanaDocker
Repozytorium

NetGuardian – SecOps Engine

Platforma do analizy logów bezpieczeństwa i detekcji anomalii. Apache Kafka jako strumień zdarzeń, WebSocket do powiadomień na żywo. Silnik reguł (YARA-like) i analiza behawioralna. Przechowuje zdarzenia w ElasticSearch (opcjonalnie). Skalowalna obsługa tysiąca zdarzeń na sekundę.

KafkaPython asyncioDuckDBWebSockets
Repozytorium

NetAegis – Orchestration MCP

Wieloagentowy system do automatyzacji konfiguracji sieci. Model Model-Controller-Protocol z agentami: NetPulse (telemetria), SecLog (audyt), NetConfig (push config). API REST pełni rolę orkiestratora, a zadania asynchroniczne realizowane przez RQ (Redis Queue). Zapewnia idempotentne zmiany konfiguracji i historię wersji.
