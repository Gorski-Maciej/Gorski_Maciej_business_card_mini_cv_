Portfolio / Zaawansowane projekty

Platformy demonstrujące umiejętności DevOps i SecOps.

Każdy projekt to kompletny stos Docker Compose z własnym API, bazą danych i dedykowaną logiką biznesową.

CloudBudget – Analityka kosztów chmury

Platforma FinOps do agregacji kosztów z różnych źródeł, prognozowania trendów i generowania rekomendacji oszczędności.

Architektura: FastAPI + PostgreSQL (dane trwałe) + DuckDB (szybka analityka w pamięci). Asynchroniczne przetwarzanie zadań przez Celery i RabbitMQ. API przyjmuje dane kosztowe, zapisuje je, a następnie cyklicznie przelicza agregaty i prognozy.
Moja rola: zaprojektowałem modele danych, zbudowałem warstwę API, skonfigurowałem kontenery Docker i zintegrowałem kolejkę zadań.
PythonFastAPIPostgreSQLDuckDBCelery
GitHub

InfraFlow – Monitoring i obserwowalność

System zbierania metryk hostów i zdarzeń infrastrukturalnych z wizualizacją w Grafanie i alertami.

Architektura: FastAPI + TimescaleDB (metryki czasowe) + Redis (kolejka). Agenci wysyłają dane do API, które zapisuje je w bazie. Prometheus scrapuje metryki, a Grafana wyświetla dashboardy.
Moja rola: stworzyłem endpointy API, skonfigurowałem Prometheus i Grafana, napisałem agenta monitorującego CPU/RAM/Dysk.
PythonFastAPITimescaleDBRedisGrafana
GitHub

NetGuardian – Bezpieczeństwo sieciowe

Platforma SecOps do detekcji zdarzeń sieciowych, analizy behawioralnej i automatycznej reakcji na incydenty.

Architektura: FastAPI + Kafka (strumieniowanie zdarzeń) + TimescaleDB (logi) + DuckDB (szybkie zapytania). WebSocket do powiadomień na żywo. Reguły bezpieczeństwa i integracja z GeoIP.
Moja rola: zaprojektowałem przepływ zdarzeń, skonfigurowałem Kafkę, zaimplementowałem analizę reguł i dashboard operacyjny.
PythonKafkaTimescaleDBDuckDBWebSocket
GitHub

NetAegis – Orkiestracja bezpieczeństwa

Dwuwarstwowy system MCP z agentami do zbierania telemetrii, logów i automatyzacji konfiguracji.

Architektura: Dwa serwisy FastAPI (MCP main i operational). Trzy agenty w Pythonie: NetPulse (telemetria), SecLog (logi bezpieczeństwa), NetConfig (automatyka). SQLite dla konfiguracji, Redis do komunikacji między komponentami.
Moja rola: zaprojektowałem architekturę wieloagentową, zbudowałem API obu warstw, zaimplementowałem agentów i ich integrację.
PythonFastAPISQLiteRedisMCP
